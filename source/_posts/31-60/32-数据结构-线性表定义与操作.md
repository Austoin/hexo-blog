## 线性表定义

**线性表(Linear List)**是具有**相同数据类型**的n(n≥0)个数据元素的**有限序列**

当n=0时,称为**空表**

表示为:$L = (a_1, a_2, \dots, a_i, a_{i+1}, \dots, a_n)$

## 核心特性

- **有限性**:元素个数有限
- **同类型**:所有元素数据类型相同,每个元素占用内存空间大小一样
- **有序性**:元素之间有先后次序

## 重要术语

- **表长**:线性表中元素的个数n
- **表头与表尾**:$a_1$是表头元素,$a_n$是表尾元素
- **前驱与后继**:
  - 除表头元素外,每个元素有且仅有一个直接前驱
  - 除表尾元素外,每个元素有且仅有一个直接后继

**易错点:位序vs数组下标**
- 线性表的**位序从1开始**($a_1, a_2...$)
- C/C++数组的**下标从0开始**

## 基本操作

操作思路:**创销、增删改查**

### 创、销(初始化与销毁)

- `InitList(&L)`:初始化表,构造空线性表L,分配内存空间
- `DestroyList(&L)`:销毁线性表,释放L占用的内存空间

### 增、删(插入与删除)

- `ListInsert(&L, i, e)`:在表L的第i个位序上插入元素e
- `ListDelete(&L, i, &e)`:删除表L第i个位序的元素,用e返回删除元素的值

### 查(查找与定位)

- `LocateElem(L, e)`:按值查找,在表L中查找给定关键字值的元素
- `GetElem(L, i)`:按位查找,获取表L第i个位序的元素值

### 其他操作

- `Length(L)`:求表长,返回线性表L的长度
- `PrintList(L)`:输出操作,按顺序输出L的所有元素值
- `Empty(L)`:判空操作,若L为空表返回true,否则返回false

## 引用传参详解

**核心准则**:对参数的修改结果需要"带回来",就必须传入引用`&`

### 按值传递(不带引用)

```c
#include <stdio.h>

void test(int x) {
    x = 1024;
    printf("test函数内部 x=%d\n", x); // 输出 1024
}

int main() {
    int x = 1;
    test(x);
    printf("调用test后 x=%d\n", x); // 输出 1,修改未带回
    return 0;
}
```

### 按引用传递(带引用)

```cpp
#include <stdio.h>

void test(int &x) {
    x = 1024;
    printf("test函数内部 x=%d\n", x); // 输出 1024
}

int main() {
    int x = 1;
    test(x);
    printf("调用test后 x=%d\n", x); // 输出 1024,修改成功带回
    return 0;
}
```

### 应用到数据结构操作

1. **为什么插入/删除需要`&L`?**
   - 插入删除会改变线性表,需要把改变后的表保留下来

2. **为什么查找`GetElem(L, i)`不需要`&L`?**
   - 查找只是"看一眼",不会修改表的内容

3. **为什么删除`ListDelete(&L, i, &e)`需要`&e`?**
   - 不仅要删除元素,还要知道删除了什么,通过`&e`把删掉的值"拿出来"

## 封装的意义

1. **团队合作编程**:封装性好,别人只需调用接口,不需要管底层实现
2. **避免重复工作**:将常用操作封装成函数,复用性高,降低出错风险
3. **考研大题利器**:熟练调用基本操作,代码整洁,体现基本功扎实

## 核心考点总结

- **1个定义**:线性表是**有限**、**有序**、**同类型**的数据元素集合
- **1个易错点**:数据元素的**位序从1开始**,与C语言数组下标区分
- **1套操作思维**:遇到任何数据结构,本能反应**"创、销、增、删、改、查"**
- **1个语法重点**:搞懂C/C++中引用`&`的用法,知道何时传值何时传引用

## 经典例题

**01.** 线性表是具有n个( )的有限序列
- A. 数据表
- B. 字符
- C. 数据元素
- D. 数据项

**答案:C**

解析:线性表是具有相同数据类型的n个数据元素的有限序列。注意区分:数据项是构成数据元素的不可分割的最小单位

**02.** 下列几种描述中,( )是一个线性表
- A. 由n个实数组成的集合
- B. 由100个字符组成的序列
- C. 所有整数组成的序列
- D. 邻接表

**答案:B**

解析:判断线性表的三个关键词:① 同类型 ② 有限 ③ 序列(有先后顺序)
- A选项:"集合"中元素没有先后顺序,不满足"序列"
- C选项:"所有整数"是无穷的,不满足"有限"
- D选项:邻接表是图的物理存储结构,而线性表是逻辑概念
- B选项:100个(有限)字符(同类型)组成的序列(有序),完美符合定义

**03.** 在线性表中,除开始元素外,每个元素( )
- A. 只有唯一的前驱元素
- B. 只有唯一的后继元素
- C. 有多个前驱元素
- D. 有多个后继元素

**答案:A**

解析:线性表"一对一"的逻辑特征。除表头元素外,每个元素有且仅有一个直接前驱;除表尾元素外,每个元素有且仅有一个直接后继。题目问"除开始元素外",必定有且仅有唯一的前驱

**04.** 若非空线性表中的元素既没有直接前驱,又没有直接后继,则该表中共有( )个元素
- A. 1
- B. 2
- C. 3
- D. n

**答案:A**

解析:没有直接前驱,说明它是表头元素(第一个);没有直接后继,说明它是表尾元素(最后一个)。一个元素既是头又是尾,说明这个表里只有1个元素
